How was this tool made?

Using python scripts to develop a frequency list then another one to match the json output with a phonetic transcription dictionary the rest is web voodoo. Check my Github for credits.

How to use

There is a menu on the top right corner. It shows your score and which round you are in. At the end of each round you will get a screen showing your results. The keyboard has all the glyphs you'll ever need but you can use your own keyboard too. The button is used to get hints, please hower your mouse on it to see the current hint.
